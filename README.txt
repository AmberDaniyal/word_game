# Guess Quest

A simple command-line word guessing game built in Python.

## Description
This game allows players to guess a randomly selected word within a limited number of attempts. After each guess, the game shows:
- Correctly placed letters
- Misplaced letters
- Incorrect letters

## Installation
1. Ensure Python 3.12 is installed on your system
2. Clone the repository: https://github.com/AmberDaniyal/word_game.git
3. Navigate to the project directory: cd guess_quest

## Usage
Run the game using Python: guess_quest.py

Follow the on-screen prompts to play:
1. The game will select a random word
2. You have 5 attempts to guess the word
3. After each guess, you'll see which letters are:
   - In the correct position
   - In the word but wrong position
   - Not in the word

## Features
- Random word selection
- Letter position feedback
- Error handling for invalid inputs
- Attempt tracking

## Project Structure
- `game.py`: Main game file containing:
  - WordGame class: Core game logic
  - GameManager class: Game flow and UI
